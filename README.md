# Exploratory Data Analysis (EDA)

## Project Description
This project involves performing an In-depth Exploratory Data Analysis on the **Titanic Dataset**. The goal is to identify patterns, handle missing values, and visualize factors that influenced passenger survival.

## Key Insights
- **Gender Factor:** Females had a significantly higher survival rate compared to males.
- **Age Distribution:** Most passengers were aged between 20-40 years.
- **Data Cleaning:** Identified missing values in 'Age' and 'Deck' columns for better accuracy.

## Technologies Used
- **Python**
- **Pandas** (Data Manipulation)
- **Seaborn & Matplotlib** (Data Visualization)

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the script: `python task2_eda.py`
